CINE

Este proyecto fue creado utilizando diferentes lenguajes de programación, como HTML5 y CSS. Fue uno de los primeros proyectos que realicé para familiarizarme con las clases y su funcionamiento. Creé una página responsiva integrando varias propiedades CSS, como el overflow, posicionamiento, la opcidad entre otros. Para mejorar la organización del contenido, apliqué técnicas de Flexbox y CSS Grid. Además, por iniciativa propia, decidí agregar formularios y aplicar diversas técnicas para organizar cada una de las secciones, logrando así un buen trabajo. Este proyecto me permitió entender mejor cómo estructurar páginas web y optimizar su presentación en diferentes dispositivos, conocimientos que he aplicado en proyectos posteriores.

En este trabajo se usaron distintas librerias como Mdn web docs y W3Schools Online Web, tambien se usa GitHub para subir el proyecto y usar GitHub Page para publicarlo de manera gratuita

FIGURAS GEOMETRICAS

Se creó un repositorio en GitHub para recrear este proyecto utilizando exclusivamente el lenguaje de Java. El objetivo principal fue comprender su funcionalidad a través de la implementación de diversas operaciones y la aplicación de conceptos avanzados como las interfaces.

Cada uno de estos proyectos se aplicaron para fortalecer el conociemiento y el funcioanmiento de cada una de las tecnicas aplicadas ademas de aplicar la libreria docs.oracle.com, para mejorar el funcionamiento y el conociemiento de la misma.

VITALIA

Mi proyecto final fue un ecommerce diseñado para la venta de medicamentos naturistas. Utilicé diversos lenguajes como HTML, CSS, Bootstrap, JavaScript y Java, junto con MySQL para la base de datos. El objetivo principal fue implementar todos los conceptos aprendidos y mejorar cada función para garantizar un diseño adecuado. Para el diseño del ecommerce, utilicé Figma. Además, referencié bibliotecas como Bootstrap, MDN Web Docs y W3Schools Online Web. El trabajo se llevó a cabo bajo la metodología Scrum y se gestionó con Trello para un proceso organizado.
