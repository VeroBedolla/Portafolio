Quien soy

Mi nombre es Veronica Bedolla Carrillo, tengo 28 años y soy originaria de Puebla, México. Soy Desarrolladora Java Full Stack, graduada en Generation México. Disfruto especialmente del desarrollo tanto en el front-end como en el back-end, buscando siempre mejorar la experiencia del usuario.

Mi experiencia abarca el desarrollo de aplicaciones móviles y de escritorio, donde combino mi pasión por la programación con mi compromiso de ofrecer un servicio de atención al cliente excepcional. Me apasiona crear herramientas que simplifiquen la vida de las personas y contribuyan positivamente a la sociedad..

Mis proyectos

CINE

Este proyecto fue creado utilizando diferentes lenguajes de programación, como HTML5 y CSS. Fue uno de los primeros proyectos que realicé para familiarizarme con las clases y su funcionamiento. Creé una página responsiva integrando varias propiedades CSS, como el overflow, el posicionamiento, la opacidad, entre otras. Para mejorar la organización del contenido, apliqué técnicas de Flexbox y CSS Grid. Además, por iniciativa propia, decidí agregar formularios y aplicar diversas técnicas para organizar cada una de las secciones, logrando así un buen trabajo. Este proyecto me permitió entender mejor cómo estructurar páginas web y optimizar su presentación en diferentes dispositivos, conocimientos que he aplicado en proyectos posteriores.

En este trabajo se usaron distintas librerías como MDN Web Docs y W3Schools Online Web. Se utilizó GitHub para subir el proyecto y GitHub Pages para publicarlo de manera gratuita.
